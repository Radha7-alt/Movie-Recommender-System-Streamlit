# Movie Recommender System

[![Streamlit App](https://img.shields.io/badge/Streamlit-Live%20App-brightgreen)](https://movie-recommender-system-app-59rtj9dpbju2en58dupdpt.streamlit.app/)

A content-based **Movie Recommendation Web App** built using **Python**, **scikit-learn**, and **Streamlit**.  
The app recommends movies similar to a selected title and displays their posters using the **TMDB API**.

---

## Live Demo

**Try the app here:**  
https://movie-recommender-system-app-59rtj9dpbju2en58dupdpt.streamlit.app/


# Features
- Content-based movie recommendations
- Cosine similarity for finding similar movies
- Interactive web interface using Streamlit
- Movie posters fetched dynamically from TMDB API
- Fast recommendations using precomputed similarity matrix
